Documentation

/startgame

Description: Start a new Guess Who game

Options: opponent - The user you want to play against (optional)

Example: /startgame @user

/ask

Description: Ask a question

Options: question - The question you want to ask (required)

Example: /ask "Does your character have blue eyes?"

/guess

Description: Make a guess

Options: character - The character you want to guess (required)

Example: /guess "Alice"

/characters

Description: List all characters and their attributes

Example: /characters

/pickcharacter

Description: Pick your mystery character

Options: character - The character you want to pick (required)

Example: /pickcharacter "Bob"

/profile

Description: Show your profile with XP and wins

Example: /profile

/leaderboard

Description: Show the leaderboard

Example: /leaderboard

/help

Description: Show help information

Example: /help

/howtoplay
